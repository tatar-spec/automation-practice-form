# 🚀 Automation Practice Form

## 📋 Project Description
This project automates web application testing using **Selenide** and **JUnit 5**.  
It includes tests for **drag-and-drop** functionality and hover interactions on GitHub.

## 🛠️ Technologies Used
- **Java 17+**
- **Selenide**
- **JUnit 5**
- **Gradle**
- **Selenium WebDriver**
